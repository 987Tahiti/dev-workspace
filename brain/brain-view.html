Plan du cerveau

Coordination

  • Le chef reste l’IA principale dans ce workspace.
  • Les autres IA sont des binômes de travail qui prennent le relais selon les limites de session.
  • Le gestionnaire de transfert doit détecter le dépassement de limite, marquer l’IA comme paused, sélectionner l’IA suivante ready et reprendre automatiquement quand une IA redevient disponible.

Surveillance

  • Etat ai-state.json des IA : OpenAI, Claude, Gemini
  • Stocker : usedTokens, usedSessions, maxTokens, sessionLimit, status
  • Exposer un script de contrôle pour afficher l’IA active, forcer la rotation et réinitialiser l’état.

Développement

  • Android : structurer un projet Gradle/Android Studio
  • Web : choisir un framework léger (React/Next.js + Node.js / Express)
  • Windows : choix .NET SDK 8, application WinUI ou console + UI selon besoin
